# PizzaShopc
A simple cli based pizza shop game written in cpp to practice oop concepts, constructor, constructor overloading and array manipulation and also written using vim .!

##How To PLay
clone into the repository, get inside the project folder ,make the executable script executable ,then execute it to play it !.

##For Windows
*Currently not available for windows*
*You can use* ***WSL*** *to run it*

##For Linux
```bash
git clone https://github.com/Anand-004/PizzaShopc.git
cd PizzaShopc
sudo chmod +x PizzaShopc
./PizzaShopc
```

##For Developers
###Adding Dynamic **Toppings** 
simply edit the toppings array declared on top of the cpp program file to add or remove your Toppings and then Compile it to the *PizzaShopc* Executable and have fun !.
